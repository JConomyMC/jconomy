# JConomy

A one-stop `Economy` provider implementing the [Vault](https://github.com/MilkBowl/VaultAPI) and [VaultUnlocked](https://github.com/TheNewEconomy/VaultUnlockedAPI) APIs.

## Goals

The goals of this project are

1. To allow easy configuration and management of multiple arbitrary Vault currencies, and
1. To provide backwards compatibility for the "primary" currency to be used in plugins using Vault legacy.
